# CatchGame

CatchGame is a project started by a student developer, in which you "catch" creatures known as boblets.

## How To Install

1. Download a Java JDK 25 distribution of your choice.
2. Download the `CatchGame.jar` file from 'releases' tab.
3. Install Java JDK 25.
4. Run the `CatchGame.jar` file via the Java Launcher or JVM.

## Disclaimer

_**This is an original student-developed game, and is in no way shape or form supposed to resemble any other game.**_
